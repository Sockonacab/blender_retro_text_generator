It's goal was to simplify gif creation for my own project !

How to use : 

 - Open Blender 
 - 1 : Go to the scripting tab (top bar, to the left)
 - 2 : Create a new script (or directly open the file if you downloaded it)

<img width="2096" height="592" alt="image" src="https://github.com/user-attachments/assets/64b8b475-16fc-4e1e-a765-bc878ea0f533" />

 - 3 : Paste the raw file inside of the newly created document
 - 4 : Run the script

<img width="2105" height="592" alt="image" src="https://github.com/user-attachments/assets/96de6c60-2d1c-464a-833a-500e6405bfd1" />

 - 5 : Open the arrow menu
 
<img width="986" height="656" alt="image" src="https://github.com/user-attachments/assets/fe7c3553-386a-4226-b42f-a7412ef41a3f" />

 - 6 : Navigate to the retro tools section at the bottom
 - 7 : Play with the settings
 - 8 : Hit : Generate Retro Text
 - 9 : Choose output dir
 - 10 : Apply camera setting if you want

<img width="985" height="822" alt="image" src="https://github.com/user-attachments/assets/38193dac-b2d8-4c11-8615-50bb3ffa6386" />

 - Then press ctrl + f12 to render you scene


 Note that every change in the settings requier you to re generate the text see step 9
